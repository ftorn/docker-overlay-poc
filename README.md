Docker Overlay Synscan
====================

## Installation

- Install pycrypto and scapy
  - pip install pycrypto
  - git clone https://github.com/secdev/scapy/
  - cd scapy
  - python setup.py install
- modify the vars inside the synscan.py file
- Execute:
  - python ./synscan

Or use the Dockerfile :)
